nabs.plan_stubs.update_sample¶

nabs.plan_stubs.update_sample(sample_name, path, n_shots)[source]¶

Update the current sample information after a run.

Updates the status values of each target in the sample, from False to True to indicate that it is shot.

Parameters

  • sample_name (str) – A name to identify the sample grid, should be snake_case style.

  • path (str) – Path to the .yml file. Defaults to the path defined when creating this object.

  • n_shots (int) – Indicates how many targets have been shot.
