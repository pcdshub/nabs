nabs.plan_stubs.get_sample_targets

nabs.plan_stubs.get_sample_targets(sample_name, path)[source]

Get the xx and yy target information from a saved sample.

Given a sample name, get the x, y grid points that are mapped for that sample.

Parameters:

  • sample_name (str) – The name of the sample to get the mapped points from. To see the available mapped samples call the mapped_samples method.

  • path (str, optional) – Path to the samples yaml file.

Returns:

``xx``, ``yy`` – Returns two lists of dictionaries, with information about the targets.

Return type:

tuple
