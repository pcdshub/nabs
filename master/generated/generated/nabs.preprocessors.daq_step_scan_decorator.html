nabs.preprocessors.daq_step_scan_decorator

nabs.preprocessors.daq_step_scan_decorator(plan)[source]

Decorator to turn a plan function into a standard LCLS DAQ step plan.

This adds the standard DAQ configuration arguments events, duration, record, and use_l3t onto the plan function and wraps the plan in the daq_step_scan_wrapper to properly execute a step scan.

See daq_step_scan_standard_args for argument specifications for the standard DAQ configuration arguments.

Parameters:

plan (plan) – A bluesky plan that yields bluesky Msg objects.

Returns:

daq_step_plan – The same plan as before, but modified appropriately for DAQ use. This will be a callable generator function.

Return type:

plan
